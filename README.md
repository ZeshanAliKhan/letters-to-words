# Letters to Words

Letters to Words is a free browser-based tool that turns a group of letters into possible English words. It supports wildcard searches, repeated-letter matching, starts-with and ends-with filters, grouped results by length, and exact anagram mode with no backend.

## Local development

```bash
npm install
npm run dev
```

## Test

```bash
npm run test
```

## Production build

```bash
npm run build
```

## Deployment

- Build command: `npm run build`
- Publish directory: `dist`
- Works on Netlify without a backend
